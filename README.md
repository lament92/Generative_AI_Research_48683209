What dose 'git clone'do; what dose github fork do; Differences between them.
- 'git clone' simply clone(download) the repository to your own local computer, 'fork' means copy repository to your own github account. fork is for making your own online version, while clone is for working on the project locally .git clone makes a local copy on your computer, while fork makes an online copy on your GitHub account. Usually, if you want to contribute to someone else's project, you first fork it, then clone your fork to your computer, make changes, push them back to your fork, and finally create a pull request.
